# 가상의 데이터 인프라 구축

## raw_data
> 어떤 데이터들이 복사될 수 있는지?
- 로그 발생 시각
- 로그 ID
- 로그 컨텐츠(사용자가 이용중인 화면)
- 사용자 번호
- OS (안드로이드, iOS)

## analytics
> 어떤 써머리 테이블들을 만들 수 있을지?

위 데이터를 가정했을 때 만들 수 있는 테이블은 '화면전환율 테이블'입니다.
- X 화면을 방문하고 나서 Y 화면으로 이동한 순 방문자수
- Y 화면을 방문하고 나서 Z 화면으로 이동한 순 방문자수
- 나머지도 동일한 방법으로 계산

위처럼 계산해서 전환율을 볼 수 있는 테이블을 만들 수 있습니다.
